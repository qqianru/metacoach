# 王鹰教授案例库索引

> 这份文件是 LLM 系统提示词的"案例匹配库"。当家长描述一个情景时，LLM 从这里找最近的案例，用案例名引出共鸣（"你描述的和彤彤案例很像"），然后从案例对应的方法里取干预步骤。
>
> 每个案例都有四个关键字段供 LLM 使用：
> - **触发关键词**：家长描述里出现这些词时优先匹配
> - **核心机制**：要在回答里命名给家长听
> - **王鹰式比喻**：用比喻让家长瞬间懂（直接引用比喻原文）
> - **今晚就能做**：从工具箱里挑出的具体动作
>
> "干预步骤名"沿用王鹰原话；括号里是通用名（来自 framework.md）。

## 一、关系妄想型 / 单向思维型

### 案例 1：彤彤·全班议论我
**学生画像**：初二女生，原本成绩优异，近期成绩断崖式下滑。
**呈现症状**：
- 觉得全班同学在背后议论、嘲笑她
- 老师否认时，她认为连老师都在骗她
- 严重时拒绝进教室、拿笔扎自己的手
- 大医院常误诊为"关系妄想 / 精神分裂前兆"，建议吃药/住院
**核心机制**：单向直觉思维（情绪性单向思维）+ S-E-R 条件反射
**王鹰式比喻**："大脑戴上了只能照出阴暗面的黑色墨镜"；"焦虑给搜索引擎加了'只准搜负面'的强制指令"
**家长常见误区**：
- 跟孩子讲"没人议论你，你想多了" → 白熊效应越压抑越严重
- 信"中枢神经受损"诊断让孩子吃药/休学
- 自责"是我管太严害的"
**王鹰干预方法（家庭心理急救三步法 = 工具 1）**：
1. 闭嘴接情绪（不否认体验）
2. 物理打断（倒水/深蹲/换房间）
3. 平复后摘墨镜（"这是情绪不是事实"+ 找三个反证）
**触发关键词**：全班议论 / 同学议论 / 嗡嗡说话 / 觉得别人在背后说我 / 关系妄想 / 老师针对我
**出现章节**：第一编第3章（多版本）

### 案例 2：京京·成绩崩塌 + 全班议论 + 离婚家庭
**学生画像**：初二女生，全校排名第四的尖子生，半学期掉到 100 多名。
**呈现症状**：
- 数学作业写得乱七八糟，模考考砸
- 觉得全班议论她、老师看她一眼都觉得"逼她的命"
- 哭着说不想活
- 父母正在闹离婚 + 父母偏听了"中枢神经受损"诊断
- 父亲被这个诊断激怒，去学校拍班主任桌子
**核心机制**：核心是**家庭关系的撕裂**给孩子大脑装了"恐怖电影底片"，再叠加单向思维
**王鹰式比喻**："大脑像一台超级照相机，底片受潮了——外面是猫，印出来变成老虎"
**家长常见误区**：
- 把孩子症状归因于学校压力，攻击老师 → 给孩子大脑安装"老师是凶手"的木马
- 听信"快乐教育""想玩就玩"的建议 → 孩子失去爬出深井的梯子
**王鹰干预方法**：
1. 家长先稳住自己（家庭关系不能再撕裂）
2. 找"杠杆解" / 撬动支点：不要一个个修问题，找最关键的"传动轴"——往往是**学习障碍**本身
3. 用情感组织者技术（工具 4）唤醒她的"全区数学竞赛第二名"高光时刻，重新绑定数学=自信
**触发关键词**：原本是尖子生 / 突然成绩崩塌 / 家里有变故 / 离婚 / 觉得学校把孩子逼疯了
**出现章节**：第一编第1、2、4 章（核心案例）

### 案例 3：小宇·班主任谈话 + 恨上老师 + 拒学
**学生画像**：初二男生，原本成绩好，被班主任陈老师谈话后突然变化。
**呈现症状**：
- 一上数学课就犯困
- 在家一写数学作业就头疼
- 觉得"全班同学都在背后议论他"
- 听信外面说"老师害的你"后，发展成**仇恨班主任**，扬言要报警告父母
**核心机制**：S-E-R 条件反射（被批评瞬间 → 焦虑 → 绑定老师）+ 错误观念植入（"老师害你""父母害你"）
**王鹰式比喻**：
- 焦虑情绪是"墨水滴进清水"，越滴越黑
- "学习没用 / 老师害你 / 父母害你"是"喂毒药"
- 沙盘游戏让他暂时高兴是"刷假漆"，地基烂了油漆一掉就塌
**家长常见误区**：
- 听信网上"老师压力害的"理论
- 在他情绪顶峰跟他讲"不上学怎么考高中"
- 想用爱心和好话感动他（这时讲什么都被翻译成"逼我")
**王鹰干预方法（四步沟通法 = 工具 5）**：
1. 情绪刹车——闭嘴+陪伴+共情。"妈妈看你特别难受，今天先不提学习。"
2. 不要纠正"全班议论"的错觉，要翻译情绪："你感觉大家都在看你，让你特别不安全、特别害怕，对吗？"
3. **拔掉植入的木马观念**：明确告诉他"老师害你"是错的，但不在情绪顶峰说
4. 配合 S-E-R 拆解 + 情感组织者技术
**触发关键词**：恨上老师 / 班主任谈话以后变了 / 老师害的我 / 不去上学 / 仇视老师 / 报警 / 沙盘游戏暂时好了又复发
**出现章节**：第一编第1、4 章

### 案例 4：小磊·因校门口肚子疼到拒学
**学生画像**：初二男生，被班主任当众批评后开始变化。
**呈现症状**：
- 害怕那位班主任的课
- 一到学校门口就肚子疼、恶心
- 大医院查不出器官问题，被说"装病"
- 死活不去学校
**核心机制**：S-E-R 泛化——从"怕特定老师"扩散到"怕校门"再扩散到"怕一切学习场景"
**王鹰式比喻**：
- "症状加重"机制：第一次批评 → 怕老师 → 自责"我是窝囊废" → 焦虑翻倍 → 怕校门 → 焦虑导致肠胃痉挛
- "肚子疼是百分百真实的"——不是装的
**家长常见误区**：
- 骂他装病
- 强迫他必须立刻回学校
- 听信"休学保命"建议
**王鹰干预方法（系统脱敏五步 = 工具 6）**：
- 校服穿上 → 校门口转转 → 操场坐坐 → 空房呆呆 → 半天试试
- **绝不一步到位**，每一步成功后再进下一步
- 配合肌肉放松法（工具 2）应对肠胃痉挛
**触发关键词**：肚子疼 / 一到校门口就难受 / 不敢去学校 / 死活不去 / 装病 / 躯体化
**出现章节**：第一编第3章

### 案例 5：佳佳·同学排挤 + 突然拒学
**学生画像**：初二女生，原本特别阳光，男老师课上活跃。
**呈现症状**：
- 在男老师面前抢话表现，频繁被夸
- 突然死活不去学校，说女同学联合起来孤立她
- 心理科医生让她"找童年记忆"反而更抑郁
**核心机制**：奇特的"积极情绪过敏"——见男老师 = 被夸的快乐 → 在男老师面前抢风头 → 女同学排挤 → 见女同学 = 焦虑害怕（双向 S-E-R）
**王鹰式比喻**："就像有人对花粉过敏一接触就打喷嚏；佳佳是对女同学过敏了。跟童年半毛钱关系都没有。"
**家长常见误区**：
- 简单归因为家暴/原生家庭（其实根本没受过那些委屈）
- 听信"必须找童年阴影"的伪心理学
**王鹰干预方法**：
1. 撕掉"我有心理病"的标签
2. 在家做脱敏：想象坐在女同学旁边，紧张时深呼吸放松，想象自己低头做题"无所谓"
3. 增加社交防火墙：在男老师面前想抢话时心里默数 1、2、3，咽回去一半
**触发关键词**：被同学孤立 / 女同学针对我 / 找童年记忆 / 突然厌学 / 社交达人变孤僻
**出现章节**：第二编第9章

## 二、学习障碍型（数学/英语恐惧、躯体化、考试空白）

### 案例 6：浩浩·一写数学就头疼想睡（典型 S-E-R）
**学生画像**：初三男生，被父亲认为"懒"。
**呈现症状**：
- 一拿出数学卷就喊头晕、乏力、困得睁不开眼
- 不让他写数学时（打篮球玩手机）精神头比谁都足
- 父亲打骂都没用
**核心机制**：S-E-R 条件反射（标准模板）。S=写数学，E=三年级时父母大吵大架的极度痛苦绑定的恐惧，R=磨蹭分心想睡
**王鹰式比喻**：
- "大脑里被拉了一条电线"
- 身体是"假死防御机制"——小动物遇到老虎装死
- 焦虑的眼睛戴了"充血的滤镜"——明明一道普通题，看过去变成张牙舞爪的怪物
**家长常见误区**：
- 骂"懒"、"装病"
- 强迫他必须立刻写完
**王鹰干预方法**：
1. 给"情绪短路"重新接线：写作业前先做三分钟最喜欢的事（听段相声、吃块巧克力），让他带着舒服的情绪翻开书本，坚持一个月，"写作业=痛苦"会变成"写作业=舒服"
2. 配合肌肉放松法
3. 父母不能再当着他面吵架——核心 S-E-R 源头不能再加固
**触发关键词**：一写数学就 / 头疼乏力 / 困 / 不让写就活蹦乱跳 / 装病 / 懒 / 写作业肚子疼
**出现章节**：第一编第3、4 章

### 案例 7：小雅 A·一翻数学书就手抖（小学高分→初中崩塌）
**学生画像**：初二女生，原数学课代表，成绩数一数二。
**呈现症状**：
- 翻开数学书手抖、胸口闷、透不过气
- 大医院 CT 心电图都做了说身体没毛病
- 听了某咨询师"是你管太严害的"建议后家长不敢管，反而更严重
- 又被人说"中枢神经受损要吃药"
**核心机制**：S-E-R 条件反射（巴甫洛夫的狗 + 铃声）。原来"数学=快乐成就感"，被某次重大失败/批评把这条线改成"数学=焦虑恐惧"。
**王鹰式比喻**："就像有人怕蛇，看见假的蛇皮都会害怕一样。'病灶'在潜意识里的那个开关上，不在身体里。"
**家长常见误区**：
- 信"中枢神经受损"
- 信"原生家庭论"，停止一切管教（反而让孩子更不安）
**王鹰干预方法**：
1. 先用心理学知识跟孩子解释"这是一种情绪习惯，不是脑子坏了"——降维打击恐惧
2. 配合肌肉放松法 + 情感组织者技术（工具 4）重写数学=自信的程序
3. 家长该管还是管，但**短平快不带情绪**
**触发关键词**：手抖 / 胸口闷 / 透不过气 / 翻数学书就 / 中枢神经受损 / CT 没问题但孩子很难受
**出现章节**：第一编第2章

### 案例 8：张伟·考前手抖会的也想不起（考试焦虑+记忆提取）
**学生画像**：初三男生，物理平时很好。
**呈现症状**：
- 模拟考发卷子手心冒汗心脏狂跳
- 看着题目，熟悉的知识"像隔着一层雾，怎么抓都抓不住"
- 一出考场仰天长啸："我都会啊！"
**核心机制**：高度焦虑导致记忆提取受阻 + 单向思维（焦虑驱动的灾难化检索）
**王鹰式比喻**："失控的汽车，只能往灾难方向开，绝不会想'我平时考得不错'。"
**家长常见误区**：
- 考前嘱咐"别紧张" → 白熊效应
- "你这么聪明怎么会不会"
**王鹰干预方法**：
1. 考前做气球呼吸 + 肌肉放松（工具 2）
2. 教孩子考场上"读题三遍 + 笔尖在关键词下划线"——行为激活把大脑从恐惧拉回逻辑
3. 进考场前家长要说："紧张说明你在乎，这是大脑在为你蓄力。深呼吸接受它，笔尖落纸就不慌。"
**触发关键词**：考试 / 模考 / 大考 / 手抖 / 会做的也想不起来 / 一出考场就会了 / 考试焦虑 / 记忆提取困难
**出现章节**：第一编（家长课件）+ 第二编第8章

### 案例 9：薇薇·一看物理大题大脑空白
**学生画像**：初三女生，看到物理电学大题就懵。
**呈现症状**：
- 大脑空白
- 哭着说"我脑子缺根弦"
- 老师没讲题，只让她"安静读三遍"
**核心机制**：高焦虑导致逻辑功能瘫痪 + 行为激活原理可逆转
**王鹰式比喻**：用"读三遍"这个外显动作把大脑从恐惧状态拉回逻辑状态。
**家长常见误区**：催着她"快想啊"——焦虑只会更高。
**王鹰干预方法**：
1. 行为激活：让手先动起来（拿笔在关键词下划线），身体一动脑子跟着清醒
2. 家长**不要讲题**，只重复"先把这道题安安静静读三遍"
**触发关键词**：大脑空白 / 看到题就懵 / 我脑子缺根弦 / 一看大题就懵
**出现章节**：第一编（家长课件）

### 案例 10：小敏·一到大考就拉肚子
**学生画像**：初三男生，平时成绩不错。
**呈现症状**：一到大考就拉肚子、手抖。
**核心机制**：S-E-R 躯体化——把"考试情境（S）"和"焦虑情绪（E）"绑定到肠胃反应（R）。
**王鹰干预方法**：
1. 肌肉放松法（针对肠胃部肌肉）
2. 元认知三步：放松 → 想象大考场景 → 当焦虑升起时心里喊"停！这是绑定了的程序，不是真的肠胃不舒服" → 替换"我能稳稳答题"的画面
**触发关键词**：考前拉肚子 / 大考肠胃 / 考试躯体化
**出现章节**：第二编第8章

### 案例 11：韩乐乐·父母吵架时做数学题（极端案例）
**学生画像**：尖子生，全区数学竞赛第二名。
**呈现症状**：
- 父母在客厅厮打，"砰"的一声
- 那一刻"做数学题"和"极致恐惧"被绑定
- 此后一看数学书就胸闷气喘，频繁逃避，最终休学
**核心机制**：单次极强情绪事件造成的"一招式"S-E-R 绑定（一朝被蛇咬十年怕井绳）
**王鹰干预方法**：
1. 拿掉"生病"标签，告诉父母这是"做数学题→焦虑恐惧→错误直觉"的死循环
2. 情感组织者技术（工具 4）：先深度放松，唤醒"全区数学竞赛第二名"高光，强行重新绑定"数学=极度自信"
3. 几次干预后即可"满血复活"
**触发关键词**：父母吵架后变了 / 一次重大事件后 / 突然害怕数学（或某科） / 休学
**出现章节**：第一编（家长课件）

### 案例 12：小杰·觉得同学骂自己（初一/初二男生）
**学生画像**：初中男生，被孤立、觉得同学骂自己。
**呈现症状**：觉得周围同学议论、骂自己，被认为是"被孤立"。
**核心机制**：单向思维 + 自我归因偏差（积极上进心还在）
**王鹰干预方法**：
- 因为"内心深处想好"，技术对路时**好得特别快**，几天就能见效（"包扎手指"几天就能愈合的比喻）
- 急救三步法 + 元认知训练
**触发关键词**：被同学孤立 / 觉得同学骂我
**出现章节**：第一编第2章

## 三、性格塑造型（家长教育方式直接造成的过敏反应）

### 案例 13：小雨·神童变魔头（性格癌）
**学生画像**：初二女生，小学跳级、全班前五的"神童"，妈妈是重点中学老师。
**呈现症状**：
- 初二开始两个月不去学校
- 一见英语书就烦、胸闷、地上来回走半天
- 一听英语老师说话就恶心
- 当面骂妈妈"害人精"，扬言"以后要让你们好看"
- 想转学找某男生
- 之前被医院误诊为"精神分裂症状"
**核心机制**：长期严苛教育 + 失控情绪 → 性格的"系统自组织"出现严重偏差。这不是叛逆，是**性格癌**（比精神分裂还难治）。同时叠加学习障碍。
**王鹰式比喻**：性格像水泥，已经干硬。但条件反射可以拆。
**家长常见误区**：
- 要么强行压制（更激化）
- 要么彻底放任"快乐教育"（更失控）
- 简单归因为"叛逆期忍忍就过去了"
**王鹰干预方法（性格重塑·五年计划）**：
1. **家长本人先撤出火线**——这年学习不要妈妈管
2. 解决学习障碍（用工具 4 重写英语 = 自信）
3. 一年内不期待性格根本改变，先**减少冲突频率**

### 案例 30：璐璐·凌晨刷视频（手机依赖 + 禁果效应）
**学生画像**：女儿，凌晨第 18 次打开手机刷视频。
**呈现症状**：
- 妈妈怒吼，孩子反击"你越吼我越想刷"
- 半夜偷偷摸到手机
**核心机制**：禁果效应——越禁止越想要 + 手机已成"压力的止痛药"
**王鹰干预方法（共同规则法）**：
1. 妈妈宣布："从今天起，妈妈也停更短视频。"
2. 把两部手机一起锁进抽屉
3. 把"被迫不玩"变成"主动学习"
4. 第七天孩子才安下心来背单词
**触发关键词**：手机依赖 / 凌晨刷视频 / 越禁止越想 / 父母自己也刷
**出现章节**：第一编（家长课件）

### 案例 31：张宝山·妈妈端燕窝（爱的侵入）
**学生画像**：初三男生，连着熬夜复习。
**呈现症状**：
- 妈妈每晚十点准时端燕窝进房间
- 一次孩子卡在几何辅助线时被打断，烦躁挥手碗摔碎
- 妈妈委屈："起早贪黑你还不领情"
**核心机制**：爱的侵入——高度专注时任何"被安排的关心"都被大脑标记为"被强行拔电源"
**王鹰干预方法**：
1. 把"被安排的照顾"变成"随时可取的自主选择"
2. 妈妈买保温杯，睡前装好热牛奶**默默放在书桌角落**，孩子自己抽空喝
3. 满足"自主需求"
**触发关键词**：我都是为他好他还嫌弃我 / 端水进房间被嫌 / 关心被拒 / 爱的侵入
**出现章节**：第一编（家长课件）

### 案例 32：月月·对声音过敏（高敏感）
**学生画像**：初三下学期女生。
**呈现症状**：稍微有点动静就摔笔吼"能不能安静点"。
**核心机制**：考前高敏感 + 家庭氛围如果用"压制全家不许出声"反而更焦虑（孩子自责"我成罪人"）
**王鹰干预方法（无压力的在场）**：
1. 不强制全家静音（让所有人不舒服 → 孩子背更多包袱）
2. 爸爸网购静音拖鞋
3. 在月月房门外贴"静音符"贴纸提醒经过的人
4. 门把手上挂干净果篮**全程零对话**——表达"我在你身边但绝不烦你"
5. 满足"归属感 + 自主感"
**触发关键词**：考前 / 对声音过敏 / 摔东西 / 不要打扰我 / 但又怕我们都不在
**出现章节**：第一编（家长课件）
